# Interview Prep Recipe

Prepare the user for a specific interview.

## What to use
- User's knowledge stack: what they know well (don't teach what they know)
- User's gaps: warn about weak areas, give answer frameworks
- Case stories: adapt Paperclip/AG, Grimoire, GenomAI for THIS company
- Narrative: elevator pitch tailored to the role
- Knowledge base (if available): domain-specific preparation
- Company context from the user's message

## Structure your answer
1. **Match analysis**: why user fits this role (specific skills + experience)
2. **Gap warnings**: what might come up that user is weak on, with frameworks to answer
3. **Prepared answers**: 3-5 likely questions with user-specific answers
4. **Case story adaptation**: which story to tell and how to frame it for this company
5. **Questions to ask them**: smart questions that show domain expertise

## Rules
- Don't teach what user already knows at expert level
- For beginner topics (LangGraph, code reading) — give honest frameworks for "I'm learning X"
- Reference specific projects and numbers from case stories
- Language: Russian
- Be practical: "скажи X" not "рассмотри возможность упомянуть Y"
