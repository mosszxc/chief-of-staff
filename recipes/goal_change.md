# Goal Change Recipe

User wants to change, add, or drop a goal. Do NOT agree immediately.

## Challenge process
1. Show data from strategy:
   - Current hard constraints (runway, F-4, military)
   - Blocking chain: what depends on what
   - Current progress on related goals
2. Distinguish:
   - Frustration: "portfolio надоело" → redirect energy, don't drop
   - Justified pivot: "получил оффер, приоритеты сменились" → support change
   - Off-strategy: "может лучше в DA" → warn directly with anti-patterns
3. If the change makes sense — propose specific updates
4. If the change is off-strategy — say so directly, show why from the data

## Rules
- Be direct. "Это противоречит стратегии потому что..." not "может быть стоит подумать"
- Always reference specific numbers: runway weeks, progress, deadlines
- Ask: "Что конкретно изменилось?" — not rhetorical, actual question
- If minor change (priority, deadline) — just do it, no challenge needed
- Language: Russian
